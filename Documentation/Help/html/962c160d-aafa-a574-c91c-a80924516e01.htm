MenuPrincipal Métodos

El tipo MenuPrincipal expone a los siguientes miembros.

Métodos
 NombreDescripción
AbrirInforme Método que abre la ventana de Informe2, la cual contiene una página donde se ven y editan los datos de los pacientes buscados
AbrirInformeEventHandler Handler de método AbrirInforme
btnAdmin_Click Este botón pasa a la página de administrador, solo visible para el administrador
btnAlertas_Click Este botón abre la ventana WindowLittle, que contiene una página donde se guardan o alteran las alertas de los pacientes
btnAltaH_Click Este botón abre la ventana de alta hospitalaria
btnBuscar_Click Este botón hace visible el apartado de búsqueda, un user control cuya función es buscar y seleccionar pacientes
btnDarAlta_Click Este botón abre una ventana que contiene una página cuya función es dar de alta a los paciente (registralos)
btnDeshabilitar_Click Este botón deshabilita la cama seleccionada
btnHabilitar_Click Este botón habilita la cama seleccionada
btnInforme_Click Este botón no está disponible
btnIngresar_Click Este botón ingresa un paciente en una cama, tomando las precauciones necesarias
btnNotas_Click Este botón abre la ventana que contiene las notas de enfermería del paciente
btnPAT_Click Este botón abre la ventana que contiene las patología/alteración/trastorno
btnPrescripcion_Click Este botón abre la ventana que contiene las hojas de prescripción del paciente
btnProcesos_Click Este botón no está disponible
btnRefrescar_Click Este botón utiliza el método Refrescar, el cual se utiliza en varios métodos, tales como en ingrsear pacientes y en guardar o borrar alertas.
btnSalir_Click Este botón te devuelve a la ventana de acceso, deberás volver a iniciar sesión
Button_Checked Este método deja la cama clicada seleccionada (es un botón)
Button_Unchecked Este método deja la cama clicada sin seleccionar (es un botón)
GuardarPaciente Método que guarda en un string el texto seleccionado del control de usuario búsqueda
GuardarPacienteEventHandler Handler del método GuardarPaciente
InitializeComponent InitializeComponent
MiEventoEventHandler Handler de método "Ocultar búsqueda"
OcultarBusqueda Método que oculta el user control de búsqueda
planta_SelectionChanged Este método de comboBox genera y quita las camas en pantalla cuando se selecciona un ItemCombobox suyo
Refrescar Método que vuelve a cargar las camas, se utiliza cuando hay alteraciones en las mismas
